>了解所有 HTTP 状态码的含义，助您更好地进行 Web 开发
## 100
*continue*
继续，客户端应继续其请求
## 101
*Switching Protocols*
切换协议。服务器根据客户端的请求切换协议
## 102
*Processing*
处理中。服务器正在处理请求，但无响应可用
## 103
*Early Hints*
早期提示。用来在最终的 HTTP 消息之前返回一些响应头
## 200
*OK*
请求成功。一般用于GET与POST请求
## 201
*Created*
已创建。成功请求并创建了新的资源
